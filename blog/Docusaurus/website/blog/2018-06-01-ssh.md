主机 host
端口 22
用户名 user
<!--truncate-->
1. 口令登录
    ```
    $ ssh [-p 22] [user@]host
    ```
    说明：
    - 默认端口22可省略
    - 用户名与本地用户名相同时可省略
1. 公钥登录
    1. 生成公钥
    ```
    $ ssh-keygen
    ```
    说明：
    - 私钥口令 passphrase，若担心私钥的安全，设置一个
    - 公钥 id_rsa.pub，私约 id_rsa
    2. 复制公钥至主机
    ```
    $ ssh-copy-id user@host
    ```
